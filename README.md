# happy-birthday-chesnok

Поздравительный сайт ко дню рождения таксы по кличке **Чеснок** (Garlic) — чёрно-подпалого пса,
которому сегодня исполняется 3 года.

«Хроники Чеснока» — тёплый кинематографичный фото-роман: hero-обложка и семь глав по чертам
характера (гурман паштета, гроза плюшевых игрушек, марафонец прогулок, чемпион по нежности,
главный соня, пловец в фонтане и единственный страх — маникюр), эпилог-поздравление и галерея
всех 17 фотографий с лайтбоксом.

## Стек

Чистый статический HTML/CSS/vanilla JS, без сборщиков. По CDN подключаются GSAP + ScrollTrigger
(reveal-анимации и параллакс) и canvas-confetti. Все фотографии — в `assets/photos/`
(`full/` — крупные ~1440×1920 для hero/глав/лайтбокса, `thumb/` — для сетки галереи).

## Локальный запуск

```bash
cd happy-birthday-chesnok
python3 -m http.server 8000
# открыть http://localhost:8000/
```

## Деплой

GitHub Pages через `.github/workflows/pages.yml` (публикует весь репозиторий как есть,
Jekyll отключён файлом `.nojekyll`). После включения Pages сайт доступен по адресу
`https://<user>.github.io/happy-birthday-chesnok/`.
